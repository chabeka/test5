Titre de ma page HTML : index.html

Cette page dispose d'un contenu html complet et bien formé avec les balises <html> <head> ou <body>

Dans le code source de la page vous ne devriez pas avoir plusieurs balises <html> <head> ou <body>

Dans les balises <head> vous devriez trouver au moins ce contenu (la balise <title> ne doit être présente qu'une seule fois) :

<meta http-equiv="Content-Type" content="text/html; charset=ISO-8859-1">
<title>Titre de mon navigateur defini depuis ma page index.html</title>
<meta name="author" value="monPremierDevJava"/>
